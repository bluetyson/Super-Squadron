# Super-Squadron
Utilities for the Super Squadron role playing game

## Recent Improvements
See [IMPROVEMENTS.md](IMPROVEMENTS.md) for a detailed list of code cleanup, bug fixes, and robustness improvements.

## Features
- Character power system with 77+ different powers
- Dice rolling utilities for game mechanics
- Character statistics and origin generation
- Support for devices, augmentations, and special abilities
